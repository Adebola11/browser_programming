# Lecture 04 - JavaScript Basics

## Features Implemented

I implemented **2 features** as required:

### Feature A: Theme Toggle (Light/Dark Mode)
- **What it does:** Toggles between light and dark color scheme
- **How to test:** Click the "Dark Mode" button in top-right corner
- **Expected result:** Page colors invert (dark background, light text), button changes to "Light Mode"

### Feature B: Click Counter
- **What it does:** Counts how many times you click a button
- **How to test:** Click the "Click Me!" button in the "Visitor Counter" section
- **Expected result:** Number increases with each click, logged to console, milestone alert at 10 clicks

### Feature C (Bonus): Quick Contact Actions
- **What it does:** Shows contact info via alert popup
- **How to test:** Click "Show Email" or "Show LinkedIn" buttons
- **Expected result:** Alert popup with contact information, logged to console
